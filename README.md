# Exo_Api
Exercice de réalisation d'une API avec différentes fonctionnalités :
- Route public /info
- Route d'authentication /auth
- Route protégée /whoami
- Route admin /rename

## Installation 
- npm install

## Configuration
Créer un fichier .env avec votre token jwt:
- JWT_SECRET="ASecretToken"

## Lancement
npm start
