# Exo_Api
Exercice de réalisation d'une API avec différentes fonctionnalités :
- Route public /info
- Route d'authentication /auth
- Route protégée /whoami
- Route admin /rename

## Installation 
- npm install express
- npm install fs
- npm install dotenv
- npm install jsonwebtoken

## Configuration
Créer un fichier .env avec :
JWT_SECRET="ASecretToken"

## Lancement
npm start
